# Linux Server Hardening Toolkit

A production-ready Bash toolkit to automate Linux server hardening for Ubuntu and Debian servers.

## Features

- SSH Hardening
- UFW Firewall Configuration
- Fail2Ban Installation
- Automatic Security Updates
- Kernel (sysctl) Hardening
- Security Audit

## Supported Operating Systems

- Ubuntu 22.04+
- Ubuntu 24.04+
- Debian 12+
- Debian 13+
